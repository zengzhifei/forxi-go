# Forxi - 探索无限可能，创造美好未来

Forxi 是一个集文件预览、图片处理、IT 工具于一体的多功能平台，让你的工作更加高效便捷。

## 我们的故事

Forxi 诞生于一个简单的想法：让日常工作更简单。我们相信，工具应该为人服务，而不是让人适应工具。因此，我们致力于打造一系列实用、易用、安全的在线工具，帮助你更高效地完成工作。

## 核心功能

### 在线文件预览
支持预览 docx、excel、ppt、jpg 等多种文件格式，无需安装任何软件。只需上传文件，即可随时随地查看内容，省去下载和安装专业软件的麻烦。

### 在线图片处理
提供图片压缩、裁剪、尺寸修改、格式转换等功能。所有处理过程都在浏览器本地完成，你的图片不会上传到服务器，真正保护隐私和数据安全。

### IT 工具集合
汇集了多种实用的编码转换和处理工具：
- Base64 编解码
- JSON 格式化
- URL 编码转换
- 颜色格式转换
- UUID 生成器
- 更多工具开发中...

### JetBrains 工具
专为技术开发者打造的实用工具，更多功能敬请期待。

## 我们的优势

### 简单易用
界面简洁直观，操作方便快捷，无需复杂的学习过程。只需选择工具，输入内容，即可获得结果。

### 功能强大
提供多种实用工具，满足各种日常工作需求。无论你是开发者、设计师还是普通用户，都能找到适合自己的工具。

### 安全可靠
所有操作均在本地或受信任的环境完成，保护你的隐私和数据安全。我们不会存储你的个人文件和数据。
